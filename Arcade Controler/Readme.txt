A simple Python app (modern UI with customtkinter) to send commands to one or two Arduino devices via serial ports.

Features:
- Select COM ports, connect, and send commands (from commands.json) using buttons.
- Supports random, custom, and test commands.

Quick Start:
1. Install Python 3.9 or newer from https://www.python.org/downloads/
2. Open a terminal in this folder.
3. Install dependencies:
	pip install customtkinter pyserial
4. Run the app:
	python arduino_arcade_controller.py
